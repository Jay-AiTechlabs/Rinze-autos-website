# RINZE AUTOS - Luxury Car Dealership Website

A modern, professional website for RINZE AUTOS LIMITED, a premium luxury car dealership based in Abuja, Nigeria. The site showcases the company's extensive inventory of foreign and local vehicles, services, and contact information.

## Features

- **Responsive Design**: Fully responsive layout optimized for mobile, tablet, and desktop devices
- **Luxury Aesthetic**: Dark theme with gold accents reflecting premium brand positioning
- **Featured Inventory**: Showcase of premium vehicles with pricing and specifications
- **Service Highlights**: Display of key services including financing, swap deals, and international imports
- **Contact Integration**: Multiple contact channels including phone, email, and location information
- **Modern UI**: Built with React components and Tailwind CSS for a polished user experience

## Technology Stack

- **Frontend Framework**: React 19
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui
- **Routing**: Wouter
- **Icons**: Lucide React
- **Build Tool**: Vite

## Project Structure

```
client/
  public/           Configuration files (favicon, robots.txt)
  src/
    pages/          Page components
    components/     Reusable UI components
    contexts/       React context providers
    hooks/          Custom React hooks
    lib/            Utility functions
    App.tsx         Main app component with routing
    main.tsx        React entry point
    index.css       Global styles and design tokens
server/             Express server for production
```

## Getting Started

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

The development server will start at `http://localhost:3000`

### Build

```bash
pnpm build
```

### Production

```bash
pnpm start
```

## Design System

The website uses a luxury minimalism design philosophy with the following color scheme:

- **Primary Background**: Deep charcoal (#1a1a1a)
- **Accent Color**: Gold/Amber (#d4af37)
- **Secondary Accent**: Slate blue (#3b4a5c)
- **Text**: Off-white (#f5f5f5)

Typography includes:
- **Display Font**: Playfair Display (headings)
- **Heading Font**: Poppins (section headers)
- **Body Font**: Inter (body text)
